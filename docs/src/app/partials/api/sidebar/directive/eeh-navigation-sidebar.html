eeh-navigation-sidebar

  1. - directive in module

This directive adds a sidebar, based on the Twitter Bootstrap navbar component, to the template. If Angular UI Router is used (which is recommended), then the sidebar directive should wrap a ui-view element. It should also be in a template that is at or near the top of the state hierarchy.

Directive Info

  • This directive executes at priority level 0.

Usage

  • as element: 
    <eeh-navigation-sidebar
  [menu-name=""]
  [nav-class=""]
  [top-offset=""]
  [menu-item-collapsed-icon-class=""]
  [menu-item-expanded-icon-class=""]
  [sidebar-collapsed-icon-class=""]
  [sidebar-expanded-icon-class=""]
  [search-input-icon-class=""]
  [search-input-is-visible=""]
  [sidebar-collapsed-button-is-visible=""]
  [sidebar-is-collapsed=""]>
...
</eeh-navigation-sidebar>
  • as attribute: 
    <ANY
  [menu-name=""]
  [nav-class=""]
  [top-offset=""]
  [menu-item-collapsed-icon-class=""]
  [menu-item-expanded-icon-class=""]
  [sidebar-collapsed-icon-class=""]
  [sidebar-expanded-icon-class=""]
  [search-input-icon-class=""]
  [search-input-is-visible=""]
  [sidebar-collapsed-button-is-visible=""]
  [sidebar-is-collapsed=""]>
...
</ANY>

Arguments

Param Type Details
menuName
(optional)
 string

Sets the name of the menu that the directive will render.
navClass
(optional)
 string

Sets the ng-class attribute of the top-level nav element.

(default: navbar-default)
topOffset
(optional)
 number

This attribute offsets the top position of the sidebar. It should equal the height of the navbar, or 0 if there is no navbar. This attribute should be used if the navbar's height is something different or if the navbar is not used.

(default: 51)
menuItemCollapsedIconClass
(optional)
 string

This attribute sets the icon used to indicate that a parent of a nested menu item is collapsed.

(default: "glyphicon-chevron-left")
menuItemExpandedIconClass
(optional)
 string

This attribute sets the icon used to indicate that a parent of a nested menu item is expanded.

(default: "glyphicon-chevron-down")
sidebarCollapsedIconClass
(optional)
 string

This attribute sets the icon used to indicate that the sidebar is collapsed.

(default: "glyphicon-arrow-right")
sidebarExpandedIconClass
(optional)
 string

This attribute sets the icon used to indicate that the search input is use for searching.

(default: "glyphicon-arrow-left")
searchInputIconClass
(optional)
 string

This attribute sets the icon used to indicate that the sidebar is collapsed.

(default: "glyphicon-search")
searchInputIsVisible
(optional)
 boolean

This attribute causes the search input to be shown or hidden.

(default: true)
sidebarCollapsedButtonIsVisible
(optional)
 boolean

This attribute causes the text collapse toggle button to be shown or hidden.

(default: true)
sidebarIsCollapsed
(optional)
 boolean

This attribute sets the state of the text collapse button.

(default: false)
